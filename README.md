# Dhikroh AI

A framework-guided transformation companion for Muslimah women, 
built on the Becoming Blueprint methodology and powered by 
Notion MCP + Claude.

## What It Is
Dhikroh AI delivers a proprietary six-stage inner transformation 
framework through intelligent conversation. It reads from and 
writes to a structured Notion knowledge base via MCP — functioning 
as both a transformation companion and a living, growing 
methodology engine.

## How It Works
- Claude Desktop connects to Notion via MCP
- Seven Notion databases form the knowledge backbone
- Session Intelligence database stores memory across conversations
- Architect Mode allows the creator to update the knowledge base 
  through natural conversation

## Three Modes
- Open Companion — free-flowing transformation conversations
- The Becoming Journey — structured stage-by-stage progression
- Architect Mode — creator knowledge management layer
